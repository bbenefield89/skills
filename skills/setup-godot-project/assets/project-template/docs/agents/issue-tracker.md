# Issue tracker

Record where issues live and the exact read/write workflow used by planning and triage skills. Reconcile this file through `$setup-matt-pocock-skills`; external tracker mutations require separate approval.

When GitHub is used, record any accepted project-planning label vocabulary here. Keep planning-role labels such as `phase`, `epic`, `ticket`, and `slice` distinct from triage-state labels. Do not create or change remote labels without explicit approval.
